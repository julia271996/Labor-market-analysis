# Labor Market Analysis (Stack Overflow Developer Survey)

## Description
Conducted an Exploratory Data Analysis (EDA) on the Stack Overflow Developer Survey dataset containing over 70,000 records to identify global labor market and salary trends.

## Tools
* Python
* Pandas
* NumPy
* Matplotlib
* [Google Colab Project Link](https://colab.research.google.com/drive/1z3t92Q8IdTGFdJV_DlP3RPHQRxvXcjgd?usp=sharing)

## My Tasks
* Performed data profiling, handled missing values, and validated data types using Pandas.
* Calculated descriptive statistics (mean, median, percentiles) for developer compensations.
* Filtered and aggregated data by programming languages, remote work formats, and experience levels.
* Prepared a structured analytical report with key market insights.

## Result
* Delivered a structured analytical report with key market insights, visualizing salary distributions and identifying high-demand tech stacks.
